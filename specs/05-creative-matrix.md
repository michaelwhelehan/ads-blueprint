# 05 — Creative Matrix Generation

The matrix generator combines themes, concepts, personas, and formats into a full set of ad entries, then generates image prompts (deterministic) and ad copy (via Gemini AI).

## npm Script

```bash
npm run ads:matrix
# With custom inputs file:
npm run ads:matrix -- --inputs=path/to/custom-inputs.json
```

## Core Data Type

```typescript
export interface CreativeMatrixEntry {
  id: string;                    // "{theme.id}-{concept.id}-{persona.id}-{format.name}"
  theme: AdTheme;
  concept: AdConcept;
  persona: AdPersona;
  format: AdFormat;
  imagePrompt: string;           // Built by buildImagePrompt() — deterministic
  adCopy: AdCopy;                // Generated by Gemini AI
  imagePath?: string;            // Filename of generated PNG (set by image generation)
  status: "pending" | "generated" | "approved" | "rejected" | "uploaded";
  sourceRound?: number;          // Which round created this entry
  carriedForward?: boolean;      // True if winner carried from previous round
}

export interface AdCopy {
  headline: string;              // Max 40 chars
  primaryText: string;           // Max 125 chars
  callToAction: string;          // One of: "Learn More", "Sign Up", "Shop Now", "Download", "Get Offer"
}
```

## Status Lifecycle

```
pending → generated → approved → uploaded
                    ↘ rejected (terminal)
```

- `pending`: Entry created, no image yet
- `generated`: Image created and saved
- `approved`: User approved in review gallery
- `rejected`: User rejected in review gallery (terminal state)
- `uploaded`: Successfully uploaded to Meta

## Algorithm

```
main():
  1. Load brand config (Zod-validated)
  2. Load ad inputs (themes + personas)
  3. Filter themes by seasonal window (see spec 03)
  4. Build combinations:
     formatPool = [feed, feed, feed, feed, story]  // 4:1 ratio
     formatIndex = 0
     for each active theme:
       for each concept in theme.concepts:
         for each persona:
           format = formatPool[formatIndex % 5]
           formatIndex++
           entry = {
             id: "{theme.id}-{concept.id}-{persona.id}-{format.name}",
             theme, concept, persona, format,
             imagePrompt: "",
             adCopy: { headline: "", primaryText: "", callToAction: "Learn More" },
             status: "pending"
           }
  5. For each entry:
     a. entry.imagePrompt = buildImagePrompt(entry, brand)  // deterministic, no API call
     b. entry.adCopy = generateCopyWithAI(entry, brand)      // Gemini API call, rate-limited
  6. Save matrix to output path (round-aware)
```

## Ad Copy Generation via Gemini

### Model

`gemini-3.1-flash-lite-preview` — fast, cheap, reliable for structured JSON responses.

### Prompt Construction

See spec 06 for the full `buildCopyGenerationPrompt()` algorithm. The prompt includes:
- Brand name, tagline, selling points
- Theme name and description
- Concept hook and bullets
- Persona name and pain points
- Brand voice (tone, style, avoid)
- Explicit instruction to never mention AI
- JSON response format specification

### API Call

```typescript
const config: GenerateContentConfig = {
  responseMimeType: "application/json",
};

const result = await ai.models.generateContent({
  model: "gemini-3.1-flash-lite-preview",
  contents: prompt,
  config,
});
```

### Response Parsing

The response is parsed with fallbacks for reliability:

```
parseCopyResponse(text, fallbackHook, fallbackBullets):
  1. Try to extract JSON from response using regex: /{[\s\S]*}/
  2. If no JSON found → return fallback (concept.hook, concept.bullets joined)
  3. Try JSON.parse on the match
  4. If parse fails → return fallback
  5. Extract headline, primaryText, callToAction with fallbacks for each field
```

### Rate Limiting

1000ms between API calls (1 req/sec) via `createRateLimiter(1000)`.

## Output Location

- **Round-aware**: `output/ads/rounds/round-{N}/matrix.json`
- **Legacy**: `output/ads/matrix.json`

Detection: if `output/ads/rounds/rounds-index.json` exists, use round paths.

## Output Format

The matrix file is a JSON array of `CreativeMatrixEntry` objects:

```json
[
  {
    "id": "{theme}-{concept}-{persona}-{format}",
    "theme": { "id": "...", "name": "...", "description": "...", "concepts": [...] },
    "concept": { "id": "...", "sceneDescription": "...", "hook": "...", "bullets": [...] },
    "persona": { "id": "...", "name": "...", "description": "...", "painPoints": [...], "targeting": {...} },
    "format": { "name": "feed-square", "width": 1080, "height": 1080, "placement": "feed" },
    "imagePrompt": "{full prompt text}",
    "adCopy": {
      "headline": "{max 40 chars}",
      "primaryText": "{max 125 chars}",
      "callToAction": "Learn More"
    },
    "status": "pending"
  }
]
```
